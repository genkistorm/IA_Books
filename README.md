# IA_Books
Stormy est une application intelligente conçue pour aider un utilisateur à trouver des livres qui lui plairaient à travers une expérience conversationnelle fluide. Contrairement aux moteurs de recherche classiques basés sur des mots-clés, Stormy utilise le Traitement du Langage Naturel pour comprendre l'essence sémantique d'une œuvre.
